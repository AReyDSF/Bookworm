# Bookworm

Read Me édition 02.11.2025, 01:26

Sur la page d'accueil, la liste complète des livres enregistrés, avec affichage partiel des infos, ainsi qu'un bouton
pour ajouter un livre.

A l'ajout du livre, il est possible de donner une note, sous forme d'étoiles, uniquement si on indique avoir lu
le
livre.

Cliquer sur la card d'un livre emmène vers une page affichant plus de détails, incluant la note donnée par
l'utilisateur, et le statut "favori" à côté du titre.
Un bouton "Edit" permet de modifier les infos du livre (ainsi que la note et le favori)
Un bouton "Delete" permet de supprimer le livre, avec message demandant confirmation avant.
